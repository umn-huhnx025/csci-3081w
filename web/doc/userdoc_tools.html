User Guide - Image Processing Suite

Graphical User Interface (GUI) Tools

The GUI comes with several useful tools for editing images. The list below details all the different tools and their functionality. The tools come in three categories: drawing tools, filter tools, and image load/save tools. It also discusses the basics of the canvas window and the controls window.

Canvas and Controls

On starting the MIA Application two windows will appear.
The first is a blank white window with a title of "MIA". This is the "Canvas" window and is where images are loaded to and edited.
The second window is grey and has a title of "Controls". This is where all the editing and load/save tools are selected and manipulated. The specific tools and their functionality is detailed below.

Drawing Tools

The MIA Application has two drawing tools by default: the pen tool and the stamp tool. These are located on the left-hand side of the Controls window.

  • Pen Tool:
    The pen tool is used by clicking and dragging across the image on the canvas, drawing a red line.
  • Stamp Tool:
    The stamp tool is used by clicking on the image on the canvas. This will place a red crosshairs on the image.
  • Undo/Redo:
    The MIA Application supports undoing and redoing operations. After an operation is carried out on the canvas (either drawing, filtering, or loading an image) you can click undo to undo that operation. Likewise you can also redo an undone operation. Up to 50 undo/redo operations are supported by default.

Filter Tools

The MIA Application several filter tools by default. These are located in the center of the Controls window.

  • Shapen:
    This filter will sharpen the image in the canvas. The higher the value, the sharper the image will become.
    Valid Range: 0.0 to 100.0

  • Edge Detect:
    This filter will find all the edges in the image on the canvas and display only those edges (all other non-edges are set to black).

  • Threshold:
    This filter will round the value of each color of each pixel up to 1.0 or down to 0.0, depending on whether that pixel's value is greater than or less than the Threshold input value, respectively.
    Valid Range: 0 to 1

  • Blur:
    This filter will blur the image in the canvas to some degree, based on the input value provided.
    Valid Range: 0.0 to 20.0

  • Grayscale:
    This filter will decolorize the image, leaving on various shades of grey, depending on each pixel's color values.

  • Channels:
    This filter will increase/decrease the color values of the image. Any input above 1.0 will increase the color value for that color, and values below 1.0 will decrease the color value for that color.
    Valid Range: 0.0 to 10.0

  • Quantize:
    This filter will reduce the number of possible intensities of each color down to the input value. If you input Quantize 4 there will only be 4 possible intensity value for each color channel (red, green, blue), spaced evenly (0%, 33%, 66%, 100%)
    Valid Range: 2 to 256
