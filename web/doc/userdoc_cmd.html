User Guide - Image Processing Suite

Using the Command Line Interface (CLI)

The command line interface is a powerful tool that can be used to automate image editing, which is especially useful when you need to edit a whole stack of images (such as successive images from an MRI or CT scan). Below we will show you how to use the command line to step through and edit up to 1000 images automatically, with only a few simple commands from the command line.

Step-by-step instructions

  1. Navigate to MIA Directory:
    To begin, open up a terminal or command line (depending on your operating system) and navigate to the location where you installed your Image Processing Suite. A command on an example machine may look like:

      C:\ > cd Program Files\Image Processing Suite\

    on Windows or

      machine-name:~/ cd Image Processing Suite/

    on a UNIX machine.
    The result should be that the command line now displays:

      <path>/Image Processing Suite:

    where <path> is the system path to your directory.

  2. Enter Commands:
    From here, you can enter any valid command to edit photos. The command line supports all filters available in the GUI and loading/saving images. Additionally, the command line has a command that allows you to apply the same filter(s) to many images at once.
    Before editing the images, create a new directory/folder in the Image Processing Suite/resources/test-images directory called "newbrainstack". This is where we will output all our files. Now, to edit the whole sample brain CT scan stack, simply type:

    ./build/bin/MIA resources/test-images/brainstack/mrbrain-8bit001.png -blur 0.6 -quantize 4 resources/test-images/newbrainstack/mrbrain-8bit001.png

    This command is pretty long, so let's break it down into parts:
    • ./build/bin/MIA
      This command starts up the MIA application, just like clicking the icon in a folder.

    • resources/test-images/brainstack/mrbrain-8bit001.png
      This is the input file, mrbrain-8bit001.png. The stuff before that is where the file is relative to the root directory of the application. (You could put in any input file from your system here, navigating around the same as when you navigated to the Image Processing Suite directory)

    • -blur 0.6
      This command tells the MIA Application that you would like to blur every image in the stack with a value of 0.6.
      (See the list of commands for all valid commands)

    • -quantize 4
      This command tells the MIA Application that you would like to quantize every image in the stack with value of 4 bins.
      (See the list of commands for all valid commands)

    • resources/test-images/newbrainstack/mrbrain-8bit001.png
      This is the output file. NOTE: This output file uses the new directory "newbrainstack" as the file where the edited images will be stored!. All the images from the input directory that match the naming of the input file will be output in the new directory with the same name.
      (See the list of commands for some nuances about editing stacks of images)


That's it! You now have all 100 edited CT scans in your new directory, and all you had to do to get them was enter a single command with the command line interface. Continue on to the next session with the link below to read more about all the available commands and their proper use.
