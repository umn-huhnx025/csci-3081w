FlashPhoto

FlashPhoto is a basic image editing and creation application. It implements eight tools (pen, eraser, spray can, calligraphy pen, highlighter, stamp, blur, and chalk), nine filters to control blur, motion blur, sharepning, edge detection, threshold, saturation, color channels, quantization, and night vision, undo/redo functionality, and loading and saving of JPEG and PNG files.

This application is built on the GLUI library with support code from the CSCI 3081W TAs.

MIA

MIA contains a subset of functionality of FlashPhoto designed for medical imaging with a couple off added features. MIA contains a red pen and marker tool, the blur, sharepning, edge detection, threshold, grayscale, color channels, and quantization filters. It also has the same undo/redo functionality and image saving and loading capabilities as FlashPhoto.

One feature unique to MIA is handling groups of images. When one image from a sequence is loaded, you can cycle through the rest of the images in the sequence. The other feature unique to MIA is a command-line interface. The command-line interface implements functionality for each of MIA's filters, saving/loading of PNGs and JPEGs, and a compare feature that determines if two images are equivalent.
