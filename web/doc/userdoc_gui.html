User Guide - Image Processing Suite

Using the Graphical User Interface (GUI)

The graphical user interface is most useful for anazlying and editing one or a small number of images. With it you can use all the tools that come packaged with the MIA Application to load, edit, and save images. You can also load images in a directory that are numbered (i.e. a "stack" of images) and "step" through them one-by-one. However, this can be tedious, and an automated method to step through and edit many images in a stack is covered in the command-line section.

In this section we will go over the steps required to convert and segment a color brain scan using the GUI interface. These are the before (left) and after (right) images:

Step-by-step instructions

  1. Start Applicataion:
    To begin, open up the MIA application by navigating to the directory in which you installed the application. An example on a windows machine would be
    C:\Program Files\Image Processing Suite\build\bin\MIA
    or in a UNIX terminal
    machine-name:~/Image Processing Suite/buid/bin/MIA.
    The application should appear as follows:


  2. Navigate to Image Directory:
    On the right hand side of the Controls window is the Image I/O panel. This panel displays all the files and folders in the directory that the MIA application starts up in. Included in your copy of the base application is a directory a large number of CT scans; this is where our example image above is from. To get to this directory, double click on resources and then test-images. You should now be in a directory that looks like this:


  3. Load Image:
    To load an image, you must first double-click its icon. The image above is mrbrain-8bit052.png. Load this image into the canvas by double-clicking it and then clicking Load Canvas. The MIA window (i.e. the "canvas") will now display the photo. We can edit this image with the Tools (far left) and Filters panel.

  4. Modify Image:
    To convert the image, first apply a Blur fliter amount of 0.6. This has the effect of reducing any noise in the image. Then set Quantize to 4 bins and apply. Your image should now match the "After" image above.

  5. Save Image:
    Finally, you can save the image. Note that saving at this point will overwrite the original, because the name matches the original name. Instead, you can change the name in the Image text field to whatever name you wish. Once you change the name, press enter (or click somewhere outside the text field). Click Save Canvas to save the image in the current directory.
    NOTE: If you wish to save in a different directory, just navigate to the desired directory (remember ../ will take you up one level in the directory, while ./ takes you into the folder you are already in) and save the image there.

That's it! You now know how to open, modify, and save an image using the Medical Imaging Application. However, doing this manually for tens or hundreds of images would be a long and tedious task. In the next section we will explain the steps on how to automate this task using MIA's command-line interface.
