User Guide - Image Processing Suite

Introduction

The Image Processing Suite is a powerful tool that can be used for image processing tasks. Using the Medical Imaging Application (MIA) tools you can examine and modify individual images as well as "stacks" of images (such as a series of scan images taken from an MRI). This guide will show you how to use MIA's intuitive graphical application, as well as its powerful command-line interface, to import, manipulate, and export your medical images.
